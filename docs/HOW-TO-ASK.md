# HOW-TO-ASK — 怎样提问才有效

> 目标：用一句清楚的话，触发对的 Skill，拿到**可检验**的辅导，而不是长篇讲义。  
> iBrain 不存考纲大库；价值来自 **话术 + Skill 流程 + GBrain 记忆写回**。

---

## 一句话公式

```text
用 <Skill>：<科目/单元> + <卡点（具体一点）> + <时间限制> + <不要什么>（可选）
```

示例：

```text
用 study-coach：AP 生物 Unit 3 光合和呼吸我串不起来，今天 25 分钟，别重讲整章。
```

---

## 好话术 vs 坏话术

| 宽泛（容易变成讲义） | 有针对性（才好用） |
|----------------------|--------------------|
| 帮我复习生物 | 用 study-coach：Unit 3 背了三个名词但串不起来，今天 30 分钟 |
| 给我出题 | 用 study-coach：特征值我会算不懂几何意义，出 1 道应用题，今天 20 分钟 |
| 考前怎么学 | 用 exam-sprint：期中还有 5 天，每天 30 分钟，Unit 3 和 7 最虚 |
| 我想做个复习 App | 用 study-coach 先诊断 2 个 gap（产物层以后再用） |
| 继续讲 | 用 session-recap：两天没学了，从 brain 接上，告诉我明天第一步 |

---

## 按场景选 Skill

| 你想做什么 | 说什么 | Skill |
|------------|--------|-------|
| 新学期 / 新课建档 | 「用 course-onboard：我选了…，先建档案」 | course-onboard |
| 不懂、要练通 | 「用 study-coach：…卡在…今天 X 分钟」 | study-coach |
| 一个误解纠偏 | 「用 misconception-fix：…我是不是理解错了」 | misconception-fix |
| 考前抓重点 | 「用 exam-sprint：还有 X 天，每天 Y 分钟」 | exam-sprint |
| 收束 / 隔天接上 | 「用 session-recap：…明天只有 Z 分钟」 | session-recap |
| 辅导自然结束 | 「好了」「今天到这」「时间到了」 | recap-reflex 自动收束 |

**默认路由**（没点名 Skill 时）：学习问题优先 study-coach；明确「建课 / 倒排 / 纠偏 / 收束」则走对应 Skill。

---

## 五个必填信息（越全越不泛）

1. **科目 / 单元** — 不要只说「生物」
2. **卡点** — 会算不懂意义 / 串不起来 / 老混成某个概念
3. **时间** — 今天 15 / 25 / 45 分钟
4. **目标形态** — 诊断 / 1 道练习 / 倒排 / 明天第一步
5. **负面约束**（可选）— 「不要模板」「别整章」「不要一堆视频」

缺信息时，Agent 应 **先问不超过 3 个短问题**，而不是开讲。

---

## 加强灵活性的半句

任意请求末尾可加：

```text
不要按固定模板回答，根据我的 brain 和今天时间灵活来。
```

---

## 辅导结束后

不必每次说「session recap」。若已开 **recap-reflex** User Rule，说一句即可：

```text
好了 / 今天就到这 / 时间到了
```

Agent 应自动收束并写回 brain。若要**只接上、不新学**：

```text
用 session-recap：我两天没学，从 brain 接上，今天不再学新的。
```

---

## Anti-patterns（学生侧）

- 「帮我全面复习」— 容易触发整章讲义
- 不说时间 — Agent 容易排满过量任务
- 同时丢 5 个单元 — 先一个单元
- 只说「讲解」不说「练」— 容易只听不练

---

## 复制即用模板

```text
用 study-coach：<科目> <单元>，我 <卡点>，今天 <分钟> 分钟，别重讲整章。
```

```text
用 exam-sprint：<科目> 还有 <天数> 天考试，每天 <分钟> 分钟，最虚的是 <单元>。
```

```text
用 misconception-fix：<概念>，我总以为 <错误想法>，帮我纠偏，30 分钟内 1 个误解就好。
```

```text
用 course-onboard：我选了 <课名>，<学期>，考试 <日期>，笔记在 <路径或无>。
```

```text
用 session-recap：今天练了 <摘要>，还卡 <点>，明天只有 <分钟> 分钟。
```
