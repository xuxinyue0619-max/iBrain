# study-coach 使用示例

两个真实学习场景，展示 iBrain 与普通 AI 的差异。

> **重要：本文档是「参考形态」，不是填空模板。**  
> Agent 必须根据学生当次输入、brain 内容和可用时间**重新设计**辅导，禁止逐段复制示例结构。

---

## 灵活原则（Agent 必须遵守）

1. **示例 ≠ 模板**  
   下列诊断表、练习、计划只是「一次可能的输出」。子概念数量、练习形式、章节标题都应随场景变化。

2. **Brain 优先于示例**  
   若 brain 里的笔记/错题与示例假设不一致，**以 brain 为准**，不要硬套示例里的「gap / misconception」。

3. **可省略任意区块**  
   - 学生只要 15 分钟 → 只给 1 个诊断结论 + 1 个练习，不要凑满六段  
   - 学生已 solid → 跳过讲解，直接迁移题  
   - 无 web 能力 → 省略资料段，不要编造链接

4. **可替换输出形式**  
   练习可以是：口述、白板、代码、实验设计、错题变式、概念地图……**不必**每次都用「练习 1 / 2 / 3」或 Feynman 卡。

5. **禁止为了「像示例」而输出**  
   - ❌ 每次都输出同样 4 列表格  
   - ❌ 每次都写「Phase 0–5」标题给学生看  
   - ❌ 把示例里的科目/知识点原样搬给不同学生  
   - ✅ 内化流程，用自然语言对话；结构服务于理解，不服务于格式

6. **写回 brain 要精简**  
   记录**本次实际诊断与下一步**，不要复制示例里的 markdown 全文。

## 示例 1：期中考前 5 天，特征值「会算但不懂」

### 背景（虚构，仅作参考）

- **科目**：线性代数
- **知识点**：特征值与特征向量
- **学生状态**：会套公式算，但不理解几何意义；期中考 5 天后
- **Brain 里可能已有**：
  - `concepts/eigenvalues.md` — 「会算 λ，不懂为什么 Av=λv」
  - `study-sessions/2026-07-01-linear-algebra.md` — 曾混淆特征向量与特征值

### 用户说什么

> 线性代数期中考还有 5 天，特征值我只会算，不懂几何意义。不要给我一堆视频链接，帮我设计能融会贯通的练习，今天只有 45 分钟。

### Agent 内部流程（不必展示给学生）

先 `gbrain search` / `query` / `get` 读 brain，再诊断 bottleneck，再设计练习。  
**不要**把 Phase 编号暴露给学生。

### 一次可能的输出（形态参考，非固定格式）

**掌握度（示例）**

| 子概念 | 状态 | 依据 |
|--------|------|------|
| 计算 2×2 特征值 | solid | 笔记/上次练习 |
| 几何意义 | gap | 学生自述 + brain 无相关记录 |
| 向量/标量混淆 | misconception | 若有历史 session 则引用，否则标「待确认」 |

**今日主线（一句话）**  
今天只打通「特征方向不变、λ 是缩放倍数」，不展开对角化。

**练习（示例形态，可换成别种）**

- 用矩阵 [[2,0],[0,3]] 先几何判断特征方向，再计算验证  
- 纠偏：「特征值是向量吗？」—— 结合学生实际错误，若无则跳过  
- 可选：30 秒口述给同学听，不强制 Feynman 标题

**45 分钟计划**  
按实际切块；若学生中途说「我几何已经懂了」，**立即减段**，不要跑完示例计划表。

**写回**  
`study-sessions/YYYY-MM-DD-eigenvalues` — 只记本次真实结论与 next_review。

### 与普通 AI 的差异

| 普通 AI | study-coach |
|---------|-------------|
| 固定讲义 + 同质题 | 诊断驱动，练什么由 gap 决定 |
| 忽视历史错误 | 有 brain 则引用，无则追问 |
| 格式千篇一律 | 结构灵活，原则不变 |

## 示例 2：课后 30 分钟，细胞呼吸「背了但串不起来」

### 背景（虚构，仅作参考）

- **科目**：生物
- **知识点**：细胞呼吸三阶段
- **学生状态**：名词能背，因果链断裂
- **Brain 里可能已有**：
  - `courses/biology-2026-spring.md` — 小测 ddl
  - `concepts/cellular-respiration.md` — 术语列表，缺因果

### 用户说什么

> 细胞呼吸我背了糖酵解、柠檬酸循环、电子传递链，但串不起来。下周小测，今天 30 分钟，帮我找出我哪里断了，别重新讲一遍课本。

### 一次可能的输出（形态参考）

**诊断重点**  
找出「术语有、因果无」的断点；若 brain 显示 O₂ 理解有误则纠偏，**若无此误解则不要硬加纠偏题**。

**练习（可替换）**

- 画物质/能量流（形式不限：列表、箭头、口述）  
- 填空 + 「为什么」：葡萄糖 → … → ATP，**空位数量随学生水平调整**  
- 1 道迁移：「只能糖酵解的细胞缺什么？」—— 可改为其他情境

**30 分钟**  
时间不够则砍掉迁移题；学生已能串起来则升级难度，不要重复示例步骤。

### 与普通 AI 的差异

| 普通 AI | study-coach |
|---------|-------------|
| 重讲整章 | 只补断链 |
| 20 张闪卡 | 1 条因果 + 少量应用 |
| 每次从零 | 写回 session 续接 |
