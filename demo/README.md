# Demo — 2 分钟看懂 iBrain 有什么用

> 这不是完整 App / 网站。  
> 这是：**学生用 Cursor + Skills 做有效复习** 的短演示入口。  
> 适合：自习缺课、要重考 AP、想少走「信息断档」弯路的人；也适合老师看「辅导怎样更有效」。

仓库主页：[iBrain](https://github.com/xuxinyue0619-max/iBrain)

---

## 别人打开这里想看什么？

| 角色 | 30 秒内应看到 |
|------|----------------|
| **学生** | 怎样用一句话开始复习；怎样留下「明天第一步」 |
| **老师 / tutor** | 怎样避免整章讲义；怎样让步骤可检验 |
| **路人** | 和普通 AI 聊天差在哪（短对比） |

---

## 三步试用（学生）

1. 打开 Cursor，确认已接 GBrain MCP（可选，没有也能先按话术练）  
2. 复制下面一句发给 Agent：

```text
用 study-coach：我明年要重考 AP（先说科目），今年自学时信息断在「上一年课堂没跟上」。
今天 25 分钟，只帮我：①标出我最可能断档的 1 个单元 ②给 1 个可检验练习 ③明天第一步。
不要整章讲义，不要模板填空。
```

3. 把对话里「诊断 / 练习 / 明天第一步」粘贴到  
   [`sessions/TEMPLATE-session.md`](./sessions/TEMPLATE-session.md) 复制出的新文件里  
   → 这就是你的复习痕迹，也是以后可以脱敏展示的素材。

更完整话术：[HOW-TO-ASK](../docs/HOW-TO-ASK.md)

## 和普通 AI 差在哪（极短对比）

| 普通复习聊天 | iBrain 交互 |
|--------------|-------------|
| 「帮我复习 AP 生物」→ 长讲义 | 先卡点 + 时间 → 1 个练习 + 明天第一步 |
| 下次从零再讲 | 写回 session / brain，可说「按上次继续」 |
| 信息断档没人标 | 明确标 ★断点（见 community 示范方法） |

详细对比形态：[EXAMPLES](../docs/EXAMPLES.md)（参考，非模板）
